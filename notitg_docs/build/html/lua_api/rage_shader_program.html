RageShaderProgram¶

Introduced in NotITG v3

  • Description

  • API reference

Description¶

An OpenGL shader program

oooh, it’s all glowy and dotty - An example of a shader taken from [FMS_Cat] Chroma - I

Any actor can have custom shaders applied to them via the Actor.SetShader() method, or by just providing Vert and/or Frag XML attributes. In addition, bits of the playfield can have shaders applied to them with Player.SetArrowShader(), Player.SetHoldShader(), Player.SetReceptorShader(), and Player.SetArrowPathShader().

Some example uses of shaders include:

  • Post processing effects (such as bloom - achieved with a ActorFrameTexture to capture a framebuffer, and a Sprite to render the framebuffer with a custom shader program)

  • Raymarching

  • Making objects shiny/reflective

  • Providing custom view/projection matrices to actors

compile(vertCode: str, fragCode: str) → void¶

  • vertCode (str) – The vertex shader code to compile, or an empty string to use NotITG’s default vertex shader

  • fragCode (str) – The fragment shader code to compile, or an empty string to use NotITG’s default fragment shader

Return type:

void

compileImmediate() → void¶

Immediately recompile/relink the shader program

Useful in conjunction with RageShaderProgram.define() or RageShaderProgram.clearDefine(), since those functions do not automatically recompile/relink the shader program.

Despite being named compileImmediate, this may not compile a shader immediately, but rather may defer compilation to when the shader is first used.

Introduced in NotITG v4

Return type:

void

define(key: str, value: str or boolean or number or nil) → void¶

Injects a #define line at the top of shader code

Since NotITG v4.2.0, this now functions with shader code that contains #version directives.

This does not automatically recompile/relink the shader program - use RageShaderProgram.compileImmediate() to do that.

Introduced in NotITG v4

Parameters:

  • key (str) – The name of the macro to define

  • value (str or boolean or number or nil) – What to define the macro as

Return type:

void
