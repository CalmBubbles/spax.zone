Song¶

Introduced in ITG

  • Description

  • Spell card format

  • API reference

Description¶

Represents a song

Spell card format¶

{
    Difficulty = int,
    Name = string,
    StartBeat = float,
    EndBeat = float,
    Color = {
        [1] = float, -- Red
        [2] = float, -- Green
        [3] = float, -- Blue
        [4] = float, -- Alpha
    }
}

GetNoteData(steps: Steps, useSeconds: boolean or nil, startBeat: number or nil, endBeat: number or nil) → list[table]¶

Returns the note data from one of the song’s steps

See Note data format

Introduced in NotITG v4

Parameters:

  • steps (Steps) – The steps to fetch note data from

  • useSeconds (boolean or nil) – If true, the returned note data will use seconds as the time measurement unit

  • startBeat (number or nil) – The start beat number (can be nil if you want to fetch all note data)

  • endBeat (number or nil) – The end beat number (can be nil if you want to fetch all note data)

Return type:

list[table]
