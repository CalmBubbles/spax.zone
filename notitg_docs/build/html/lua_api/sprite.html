Sprite¶

Introduced in ITG

  • Description

  • XML attributes

  • API reference

Description¶

A sprite is an actor with a texture

XML attributes¶

File

A filepath to the initial image the sprite should use

loop(enable: boolean) → void¶

Sets whether an animated texture should loop

This is an alias for RageTexture.loop()

Introduced in ITG

Parameters:

enable (boolean) – true to loop the animation, false to have a play-once animation

Return type:

void

looppoint(point: number) → void¶

Sets the animated texture’s loop point

This can be used to make multiple-state animations using a single spritesheet - such as having a run cycle set the loop point when starting/stopping a run.

Introduced in NotITG v4

Parameters:

point (number) – The new loop point to set

Return type:

void

texcoordvelocity(x: number, y: number) → void¶

Sets the texture scroll speed

A velocity of 1 means the texture will scroll entirely once per second - 2 makes it scroll twice per second, etc.

Introduced in ITG

Parameters:

  • x (number) – The X scroll speed to apply

  • y (number) – The Y scroll speed to apply

Return type:

void
