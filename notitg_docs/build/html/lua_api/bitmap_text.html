BitmapText¶

Introduced in ITG

  • Description

  • XML attributes

  • API reference

Description¶

A actor that renders text. Text can be aligned using Actor.horizalign() (Actor.halign() does not work).

XML attributes¶

Font

Name of the font to use (Eg: _eurostile outline).

Text

Text that should be displayed. This can be changed later with BitmapText.settext().
